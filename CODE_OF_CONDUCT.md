# Code de conduite

OpenIRN vise à rester un projet accueillant, sérieux et utile.

Les contributions doivent respecter les principes suivants :

- rester courtois ;
- discuter les désaccords techniquement ;
- ne pas publier de données confidentielles ;
- respecter les licences des contenus tiers ;
- documenter les changements significatifs.

Les comportements abusifs, discriminatoires ou agressifs n’ont pas leur place dans le projet.
