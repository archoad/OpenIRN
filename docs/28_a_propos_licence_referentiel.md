# Patch 026 — Écran À propos / Licence / Référentiel

Ce patch ajoute un écran `À propos` accessible depuis l’icône d’information de l’écran principal.

L’objectif est de préparer la publication open source d’OpenIRN en rendant visibles :

- l’identité de l’application ;
- la séparation entre le code OpenIRN, le référentiel officiel aDRI et les données locales ;
- la version du référentiel IRN utilisé ;
- la source GitLab du référentiel ;
- le fichier importé ;
- le checksum SHA-256 ;
- la licence déclarée du référentiel ;
- les éventuels avertissements d’import.

Le patch ne modifie pas le modèle de notation, les campagnes, l’export ou l’import JSON.
