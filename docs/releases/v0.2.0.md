# OpenIRN v0.2.0

Version majeure fonctionnelle du prototype OpenIRN.

## Principales évolutions

- Interface mobile responsive.
- Séparation entre accueil, évaluation et référentiel aDRI IRN.
- Campagnes synchronisées via API.
- Synchronisation automatique avec indicateur de connexion.
- Temps réel via Server-Sent Events.
- Authentification utilisateur à l'ouverture d'une campagne.
- Base utilisateurs centralisée côté serveur.
- Gestion des rôles : Administrateur, Pilote IRN, Évaluateur, Validateur, Lecteur.
- Filtrage des critères affectés aux évaluateurs.
- Interface administrateur pour modifier les codes utilisateurs.
- Simplification UX : suppression de la notion visible de campagne locale.
- Backend FastAPI enrichi : sync, status, events SSE, users, auth.
